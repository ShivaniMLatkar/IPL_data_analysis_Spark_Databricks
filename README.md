IPL Data Analysis — Spark on Databricks

End-to-end analysis of Indian Premier League (IPL) cricket data from 2008–2017 using PySpark on Databricks, with source files stored in Amazon S3. The notebook builds typed Spark schemas, transforms ball-by-ball and match data, runs Spark SQL queries against five related tables, and visualizes findings with Matplotlib and Seaborn.

Dataset
Source: IPL Data till 2017 — raghu543 on data.world
